# Misinformation-study-Masters-Thesis
For the capstone requirement for my Master of Arts (MA) in Psychological Sciences program at Montclair State University (MSU), I worked with Dr. Michael Bixter as my thesis mentor to investigate the cognitive correlates of 'vulnerability to misinformation' (VTM), a psychological construct referring to how likely an individual is to fall for misinformation found online. We set up the experiment as the broad 'cognitive reflection test (CRT)', which is a measure of one's tendency to reflect and deliberate before jumping to conclusions, as the independent variable and VTM as the dependent variable. Informed by the existing and emerging literature, we included the Conspiracy Mentality Questionnaire (CMQ) and Bullshit Receptivity Test (BRT) as two mediating factors in the model. We recruited 200 participants via Cloud Research. The survey was designed on Qualtrics. The study was pre-registered in the Open Science Framework (OSF) and approved by the IRB of MSU. The data analysis was done using R, with most work done using the Laavan package.
