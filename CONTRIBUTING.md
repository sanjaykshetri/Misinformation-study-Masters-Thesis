# Contributing Guidelines

Thank you for your interest in this research project!

## About This Project

This repository contains the research materials for a completed master's thesis. While the primary research is complete, contributions in the following areas are welcome:

### Ways to Contribute

1. **Code Improvements**
   - Enhance data visualization
   - Optimize analysis scripts
   - Add additional statistical tests
   - Improve code documentation

2. **Documentation**
   - Fix typos or clarify explanations
   - Add comments to complex code sections
   - Improve README clarity

3. **Replication Studies**
   - Share your own replication attempts
   - Suggest methodological improvements

### How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/improvement`)
3. Make your changes
4. Commit with clear messages (`git commit -m 'Add: brief description'`)
5. Push to your branch (`git push origin feature/improvement`)
6. Open a Pull Request

### Code Style

- Follow tidyverse style guide for R code
- Include comments for complex operations
- Test your code before submitting

### Questions?

Open an issue or contact the repository owner.

## Code of Conduct

Be respectful, constructive, and professional in all interactions.
